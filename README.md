 # OpenCut 中文社区项目

 ## 项目目标

 成为 OpenCut 在中国的中文用户社区入口，为中文用户提供文档、教程、汉化、模板等服务。

 ## 项目结构

 ```
 opencut-zh/
 ├── README.md          # 本文件 - 项目总览
 ├── TRACKER.md         # 跟踪文档 - 记录进度、待办、决策
 ├── docs/              # 中文文档（翻译自官方）
 ├── site/              # 中文社区落地页/导航站
 ├── content/           # 教程、视频脚本、社交媒体内容
 └── scripts/           # 自动化脚本
 ```
