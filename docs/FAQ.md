 # OpenCut 常见问题

 ## 基础问题

 ### OpenCut 是什么？
 OpenCut 是一个免费、开源的视频编辑器，定位为 CapCut（剪映）的开源替代品。支持在浏览器、桌面和移动端使用。

 ### OpenCut 真的免费吗？
 是的，完全免费。OpenCut 采用 MIT 开源协议，你可以自由使用、修改和分发。

 ### 和剪映/CapCut 有什么区别？
 - **开源**：代码完全公开，任何人都可以审查和贡献
 - **免费**：没有付费墙，所有功能免费
 - **可扩展**：新版将支持插件系统，功能可无限扩展
 - **隐私**：数据在本地处理，不会上传到云端

 ## 使用问题

 ### 如何开始使用？
 直接访问 [opencut.app](https://opencut.app)，在浏览器中即可使用，无需安装。

 ### 支持哪些平台？
 目前支持 Web 端。新版重写后将支持：
 - Web（浏览器）
 - 桌面（Windows / macOS / Linux）
 - 移动端（iOS / Android）

 ### 支持中文吗？
 目前官方版本暂不支持中文界面。我们正在建设中文社区，未来将提供汉化版本和中文文档。

 ### 项目文件保存在哪里？
 当前版本的项目文件保存在浏览器本地存储中。建议定期导出备份。

 ## 技术问题

 ### 用什么技术栈构建的？
 - **Classic 版本**：Next.js + Rust WASM + Turborepo
 - **新版（重写中）**：Rust 核心 + 插件系统 + Bun + Moon

 ### 如何参与开发？
 新版架构还在设计中，暂未开放外部代码贡献。你可以：
 - 关注 [GitHub Issues](https://github.com/OpenCut-app/OpenCut/issues)
 - 加入 [Discord](https://discord.gg/zmR9N35cjK) 了解最新进展
 - 参与中文社区的文档翻译和教程制作

 ### 新版什么时候发布？
 官方没有给出具体时间表，但开发过程是公开的，可以在 GitHub 上跟踪进度。
